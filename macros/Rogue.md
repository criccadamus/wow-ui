# Rogue macros

```txt
/stopcasting
#showtooltip Tricks of the Trade
/cast [@mouseover] Tricks of the Trade
```

```txt
#showtooltip
/cleartarget
/targetenemyplayer
/cast [harm,nodead] Sap
```

## Subtlety

```txt
#showtooltip
/cast [bonusbar:1] Shadowstrike; Backstab
```

```txt
#showtooltip
/cast Vanish
/cast Shadowstrike
/stopattack
```

```txt
#showtooltip
/cast Shadow Dance
/cast Shadowstrike
```
